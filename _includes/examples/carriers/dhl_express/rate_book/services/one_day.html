One day

Express shipment that will be delivered the next working day

Requirements:

  • supplying a pickup request at creation time of the shipment is mandatory, if you don't have regular pickups
  • the date for pickup_time.earliest and pickup_time.latest has to be the same
  • supplying a pickup_address is optional
  • supplying a description for the shipment (not as part of the package) is mandatory
  • supplying phone is mandatory for sender and recipient addresses
  • state has to be 2 characters long
