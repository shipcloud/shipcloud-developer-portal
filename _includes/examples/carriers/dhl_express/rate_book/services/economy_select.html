Economy select

The DHL Express service economy select provides you with a "door-to-door delivery within Europe via the DHL Express Road Network on a specific day".

Additional requirements (see global requirements):

  • Only available for international shipments within Europe
  • Maximum dimensions (l/w/h): 120 x 80 x 80cm
  • service has to be dhl_express_economy_select
