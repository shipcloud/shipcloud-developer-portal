Bücher- und Warenversand

If you want to send books, replacement parts, electronic devices, mobile telephone accessories, data storage devices, textiles and household goods you can do so by using the service Bücher- und Warenversand by Deutsche Post AG.

Requirements:

  • package.type has to be parcel_letter
  • carrier has to be dpag
  • service has to be standard
