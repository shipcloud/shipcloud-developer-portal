Pickup station (Packstation)

Besides sending via Deutsche Post to a customer you can also send shipments directly to a DHL Packstation.

Requirements:

  • care_of = customer id number (postnummer)
  • street has to be PACKSTATION
  • street_no = number of packstation
  • Only applicable for the following products:
    • Bücher- und Warensendung
    • Großbrief
    • Maxibrief
    • Warenpost
