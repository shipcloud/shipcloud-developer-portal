Returns

A shipment that the customer sends back to the store / sender

Notice: The carrier is working in partnership with (their) "local heroes". This means that you can't specify the carrier that will transport the return shipment. The carrier is chosen by PARCEL.ONE and the shipment will then have to be brought to one of the outlets of the corresponding carrier from the shipping label.

Requirements:

  • Returns can be send from the following countries:
    • Austria (AT)
    • Belgium (BE)
    • Germany (DE)
    • Denmark (DK)
    • Spain (ES)
    • Finland (FI)
    • France (FR)
    • Italy (IT)
    • The Netherlands (NL)
    • Sweden (SE)
  • label.format has to be pdf_a5
  • package.type has to be parcel
