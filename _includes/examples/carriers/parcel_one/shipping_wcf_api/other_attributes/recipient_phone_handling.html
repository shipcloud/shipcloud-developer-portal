Recipient phone handling

You can provide a recipients phone address using the to.phone attribute. When this is done and the carrier used for doing the last mile delivery needs the information, Parcel.ONE will provide it to them. This way the carrier is able to contact recipients if needed for doing the delivery.

Notice: Providing the information doesn't guarantee, that it will be used by the carrier. The carrier decides wether the information is needed for the delivery.
